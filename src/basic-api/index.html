CanvasRenderingContext2D API 汇总文档

实例属性

绘图状态

  • globalAlpha

    全局透明度 (0-1)。默认值：`1.0`。
  • globalCompositeOperation

    合成操作模式。默认值：`"source-over"`。
  • filter

    CSS 滤镜效果。默认值：`"none"`。

填充和描边样式

  • fillStyle

    图形内部的颜色和样式。
  • strokeStyle

    图形边线的颜色和样式。

线条样式

  • lineWidth

    线宽。默认值：`1.0`。
  • lineCap

    线端样式。可选值：`"butt"`、`"round"`、`"square"`。
  • lineJoin

    线拐角样式。可选值：`"miter"`、`"round"`、`"bevel"`。
  • miterLimit

    斜接限制。默认值：`10.0`。
  • lineDashOffset

    虚线偏移量。默认值：`0.0`。

阴影

  • shadowColor

    阴影颜色。
  • shadowBlur

    阴影模糊半径。
  • shadowOffsetX

    阴影水平偏移。
  • shadowOffsetY

    阴影垂直偏移。

文本样式

  • font

    字体定义。默认值为 `"10px sans-serif"`。
  • textAlign

    文本对齐设置。可选值：`"start"`、`"center"`、`"end"`。
  • textBaseline

    文本基线设置。可选值：`"top"`、`"hanging"`、`"middle"`、`"alphabetic"`、`"ideographic"`、`"bottom"`。

  • direction

    文本方向。可选值：`"ltr"`、`"rtl"`、`"inherit"`。
  • letterSpacing

    字母间距。默认值为：0px。
  • fontKerning

    字体字距调整。允许的值：auto（默认）、normal 或 none。
  • fontStretch

    字体拉伸。允许的值：ultra-condensed、extra-condensed、condensed、semi-condensed、normal（默认）、semi-expanded、expanded、extra-expanded 或 ultra-expanded。
  • fontVariantCaps

    字体变体大小写。允许的值：normal（默认）、small-caps、all-small-caps、petite-caps、all-petite-caps、unicase 或 titling-caps。
  • textRendering

    文本渲染。允许的值：auto（默认）、optimizeSpeed、optimizeLegibility 或 geometricPrecision。
  • wordSpacing

    单词间距。默认值为：0px。

图像处理

  • imageSmoothingEnabled

    是否启用平滑。默认值：`true`。
  • imageSmoothingQuality

    平滑质量。可选值：`"low"`、`"medium"`、`"high"`。

实例方法

路径操作

  • beginPath()

    开始新路径。
  • closePath()

    闭合路径。
  • moveTo(x, y)

    将路径的起点移动到指定坐标。
  • lineTo(x, y)

    绘制直线到指定坐标。
  • arc(x, y, r, sAngle, eAngle, anticlockwise)

    绘制圆弧。
  • ellipse(x, y, rx, ry, rotation, sAngle, eAngle, anticlockwise)

    绘制椭圆。
  • rect(x, y, w, h)

    添加矩形路径。
  • quadraticCurveTo(cpx, cpy, x, y)

    添加二次贝塞尔曲线。
  • bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)

    添加三次贝塞尔曲线。

绘制命令

  • fill([fillRule])

    填充路径。可选参数：`"nonzero"`/`"evenodd"`。
  • stroke()

    描边路径。
  • clip([fillRule])

    设置裁剪区域。

绘制矩形

  • fillRect(x, y, w, h)

    绘制填充矩形。
  • strokeRect(x, y, w, h)

    绘制矩形边框。
  • clearRect(x, y, w, h)

    清除指定矩形区域。

图像处理

  • drawImage(image, dx, dy, dWidth, dHeight)

    绘制图像。
  • createImageData(w, h)

    创建像素数据对象。
  • getImageData(sx, sy, sw, sh)

    获取像素数据。
  • putImageData(data, dx, dy, dirtyX, dirtyY, dirtyW, dirtyH)

    写入像素数据。

变换

  • translate(x, y)

    平移坐标系。
  • rotate(angle)

    旋转坐标系。
  • scale(sx, sy)

    缩放坐标系。
  • transform(a, b, c, d, e, f)

    应用变换矩阵。
  • setTransform(a, b, c, d, e, f)

    重置并应用变换。
  • resetTransform()

    重置为单位矩阵。
  • getTransform()

    获取当前变换矩阵。

渐变与图案

  • createLinearGradient(x0, y0, x1, y1)

    创建线性渐变。
  • createRadialGradient(x0, y0, r0, x1, y1, r1)

    创建径向渐变。
  • createConicGradient(startAngle, x, y)

    创建锥形渐变。
  • createPattern(image, repetition)

    创建图案。

线型

  • setLineDash([])

    设置当前的虚线样式。默认为空数组[]，表示实线。
  • getLineDash()

    返回当前线段样式的数组，数组包含一组数量为偶数的非负数字。

文本绘制

  • fillText(text, x, y, maxWidth)

    填充文本。
  • strokeText(text, x, y, maxWidth)

    描边文本。
  • measureText(text)

    返回文本尺寸对象。

状态管理

  • save()

    保存当前绘图状态。
  • restore()

    恢复保存的绘图状态。

辅助方法

  • isPointInPath(x, y, fillRule)

    检测点是否在路径内。
  • isPointInStroke(x, y)

    检测点是否在描边内。
  • getContextAttributes()

    获取上下文属性。
